# BalticGreySealIPM

Code to implement the Baltic grey seal integrated population model

The code is based on the work published in

Vanko, M., Helle, I., Kunnasranta, M., Ahola, M., Bäcklin, B.-M., Carlsson, A. M., Cervin, L., Persson, S., and Vanhatalo J. (2026). Bayesian integrated population model of Baltic grey seals (Halichoerus grypus grypus) informs on population carrying capacity and the impact of Baltic herring on reproduction. Ecological Modelling. https://doi.org/10.1016/j.ecolmodel.2026.111666
